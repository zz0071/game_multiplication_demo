# 網頁遊戲完整需求規格書


## 📋 一、遊戲核心設定

### 1.1 題目規則
- **運算類型**：乘法運算
- **題目範圍**：
  - 2位數 × 1位數（10-99 × 0-9）
  - 1位數 × 2位數（0-9 × 10-99）
  - 包含 0 和 1 的運算
- **答案限制**：乘積結果 ≤ 999
- **題目生成**：自動過濾超過 999 的組合

### 1.2 關卡設計
- **每局關卡數**：10 關
- **每關題目數**：5 題
- **總題目數**：50 題/局
- **生命值**：10 次錯誤機會（全局共用）
- **答題時間**：每題 30 秒倒數計時
- **難度設計**：關卡遞增（建議規則如下）
  ```
  關卡 1-2：  10-30 × 1-3
  關卡 3-4：  20-50 × 2-5
  關卡 5-6：  30-70 × 3-7
  關卡 7-8：  40-90 × 4-8
  關卡 9-10： 50-99 × 5-9
  ```

## 🎮 二、遊戲機制

### 2.1 計分系統
| 項目 | 規則 |
|------|------|
| **基礎分數** | 每題答對 +10 分 |
| **連續答對加分** | 建議規則：<br>• 連續 3 題：+5 分<br>• 連續 5 題：+10 分<br>• 連續 10 題：+20 分 |
| **時間加分** | （可選）剩餘時間每秒 +0.5 分 |
| **答錯/超時** | 0 分，扣 1 條生命 |

**❓ 請確認連續答對加分的具體倍數或分數**

## 🎨 三、介面與互動

### 3.1 視覺設計（雷電風格）
- **主題**：太空/射擊風格
- **玩家元素**：戰機/飛船
- **題目呈現**：敵機或隕石顯示算式
- **生命值顯示**：3 顆愛心/護盾
- **倒數計時器**：30 秒進度條或數字倒數
- **語言**：繁體中文

### 3.2 答題互動
- **答題方式**：點擊 6 個圖卡選項（選擇題）
- **圖卡內容**：
  - 1 個正確答案
  - 5 個干擾選項（需合理設計，例如：±10、±50、常見錯誤答案）
- **答題回饋**：
  - ✅ **答對**：播放特效動畫 + 音效（擊落敵機）
  - ❌ **答錯**：播放錯誤動畫 + 音效（戰機受損）
  - ⏰ **超時**：視為答錯

### 3.3 動畫效果需求
- 答對：擊落動畫 + 得分飄字
- 答錯：震動 + 生命值減少動畫
- 過關：關卡完成慶祝動畫
- 遊戲結束：統計畫面淡入

### 3.4 音效需求
- 背景音樂（可循環播放）
- 答對音效
- 答錯音效
- 遊戲結束音效
- 過關音效
